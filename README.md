# Prosthetic Hand Simulation (Terminator Style)

This is a prosthetic hand simulation modeled in SolidWorks. The project demonstrates mechanical design and assembly of a fully articulated prosthetic hand, inspired by the "Terminator" hand.

## Features
- Fully articulated fingers with realistic movement
- SolidWorks assembly simulation
